Description détaillée

Structures et Primitives de l'écran.

L'ecran est géré par les primitives de la bibliothèque "curses"
L'ecran est divise en 2 fenetres :

  • la fenetre d'affichage
  • la fenetre des messages

◆ ecran_creer()

ecran_t * ecran_creer ( const int  H_affichage,
const int  L_affichage 
)

Creation d'un ecran type "ncurses".

Paramètres
[in]H_affichage: hauteur demandée pour l'écran
[in]L_affichage: largeur demandée pour l'écran

On ne peut avoir qu'un ecran par console/terminal

  • si H_affichage = 0 --> hauteur ecran = hauteur console
  • si L_affichage = 0 --> largeur ecran = largeur console

La sortie erreur standard est redirigée dans un fichier pour afficher les erreurs dans la fenetre des messages et ne pas polluer l'affichage de l'ecran

Renvoie
la structure ecran_t instanciée
